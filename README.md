# 목적과 목표
케어-인슈어런스(care-insurance)는 보험사의 간병인 현물 보상을 위탁받아 간병인을 파견하는 비즈니스 과정을 전산화 하기 위한 ERP 제품의 백앤드입니다.

아래와 같은 기능을 제공하는것을 목표로 합니다.
- 간병인 파견 접수
- 간병 등록 및 상태 관리
- 보험사에게 간병비를 청구
- 보험사로부터 입금된 간병비 관리
- 간병인에게 간병비를 정산

# 기술스택

- [Kotlin](https://kotlinlang.org/)
- [Spring](https://spring.io/)
    - Spring Boot
    - Spring Web
    - Spring Data JPA
